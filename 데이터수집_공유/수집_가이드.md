# 한글 지문자 데이터 수집 가이드

## 1. 환경 설정

### Python 설치
- **Python 3.11** 권장 (3.14에서는 mediapipe가 지원되지 않음)
- https://www.python.org/downloads/release/python-3119/

### 패키지 설치
```bash
pip install -r requirements.txt
```

---

## 2. 프로그램 실행

```bash
python dataset.py
```

- 웹캠이 연결되어 있어야 합니다.
- 실행하면 웹캠 화면과 초록색 가이드 사각형이 표시됩니다.

---

## 3. 조작 방법

| 키 | 기능 |
|----|------|
| `m` | 자음 모드 / 모음 모드 전환 |
| `q` | 프로그램 종료 |
| 숫자/알파벳 키 | 해당 글자 녹화 시작 (10초간 자동 수집) |

## 5. 수집 방법

1. 프로그램을 실행합니다.
2. 화면의 **초록색 사각형 안에 손을 위치**시킵니다.
3. 수집할 글자에 해당하는 키를 누릅니다.
4. **10초 동안** 해당 지문자 동작을 유지합니다 (사각형이 빨간색으로 변함).
5. 녹화가 끝나면 다음 글자로 넘어갑니다.
6. 모음을 수집하려면 `m` 키를 눌러 모음 모드로 전환합니다.

---

## 6. 수집 시 주의사항

- 손이 **가이드 사각형 안**에 있어야 인식됩니다.
- 녹화 중에는 **동작을 안정적으로 유지**해 주세요.
- 한 글자당 **최소 2~3회** 반복 수집을 권장합니다 (각도, 위치를 약간씩 다르게).
- 배경이 깔끔하고 **조명이 밝은 환경**에서 수집하세요.
- 수집 중 잘못 녹화한 경우, `dataset/글자/landmarks_npy/` 폴더에서 해당 파일을 삭제하면 됩니다.
- 프로그램을 재실행해도 **기존 데이터에 이어서 수집**됩니다 (덮어쓰지 않음).

---

## 7. 수집 데이터 구조

프로그램 실행 후 `dataset/` 폴더가 자동 생성됩니다:

```
dataset/
├── ㄱ/
│   ├── landmarks_npy/    ← .npy 파일 (학습용)
│   └── landmarks_csv/    ← .csv 파일 (확인용)
├── ㄴ/
│   ├── landmarks_npy/
│   └── landmarks_csv/
├── ...
├── ㅏ/
│   ├── landmarks_npy/
│   └── landmarks_csv/
└── ...
```
